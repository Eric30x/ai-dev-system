# V10.4 — Artifact System

**Released**: June 2026

## What shipped

- ✅ Artifact System: auto-snapshot every build
  - source.zip — complete source code
  - logs.txt — execution logs
  - metadata.json — version info + model + timestamp
  - final.zip — download package
- ✅ Version History API (`GET /api/projects/:id/artifacts`)
- ✅ Artifact Download API (`GET /api/projects/:id/artifact-download?artifactId=`)
- ✅ Rollback API (`POST /api/projects/:id/rollback`)
- ✅ Version Compare API (`GET /api/projects/:id/artifacts/compare?v1=&v2=`)
- ✅ Versions modal in Workspace UI
- ✅ Worker auto-saves artifacts on every task completion
- ✅ Disk storage: `workspaces/{id}/.artifacts/v{N}-{type}`

## Artifact Types

| Type | File | Description |
|------|------|-------------|
| source | `v{N}-source.zip` | Complete source code snapshot |
| log | `v{N}-logs.txt` | Structured execution logs |
| metadata | `v{N}-metadata.json` | Version info, model, timestamps |
| zip | `v{N}-final.zip` | Final download package |
| readme | `v{N}-README.md` | Project README (if exists) |
