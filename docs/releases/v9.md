# V9 — Core Architecture

**Released**: June 2026

## What shipped

- ✅ Express API Gateway with JWT auth
- ✅ Redis + BullMQ distributed job queue
- ✅ Worker Pool with Planner + Executor agents
- ✅ LLM Router (DeepSeek via Anthropic-compatible API)
- ✅ PostgreSQL + Prisma ORM (User, Project, Task, LogEntry, Usage, ApiKey)
- ✅ Stripe billing integration (graceful mock mode)
- ✅ Project generation: Plan → Execute → Zip → Download
- ✅ Web UI with Dark Mode + Glassmorphism
- ✅ Docker Compose (Postgres + Redis)
- ✅ SSE real-time log streaming

## Architecture

```
Web UI → Express API → Redis Queue → Worker → Planner → Executor → Zip → Download
```

## Database

6 tables: User, ApiKey, Project, Task, LogEntry, Usage
