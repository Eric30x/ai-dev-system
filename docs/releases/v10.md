# V10 — Workspace + Agent Platform

**Released**: June 2026

## What shipped

- ✅ Workspace page with file tree + code editor + AI Chat + Terminal
- ✅ Monaco Editor integration (VS Code core, CDN v0.50)
- ✅ AI Chat Agent (`POST /api/workspace/:id/agent`) — actually modifies files
- ✅ Verifier Agent: npm install + entry check + package.json validation
- ✅ AI Fixer Agent: LLM diagnoses errors and generates patches (≤3 rounds)
- ✅ Enhanced LLM Router: 4-provider support (DeepSeek, Claude, OpenAI, Gemini)
- ✅ New database tables: Workspace, Conversation, Message, Artifact, Deployment
- ✅ Project Workspace API: file tree, read/write files, SSE stream, chat
- ✅ Self-healing pipeline: Plan → Execute → Verify → Fix → Verify → Ship
- ✅ Template Gallery: SaaS Dashboard, Blog CMS, AI Agent, E-Commerce, CRM, REST API
- ✅ Command Palette (Ctrl+K)

## Architecture

```
Web UI → API → Redis Queue → Worker
                                ├── Planner (LLM)
                                ├── Executor
                                ├── Verifier
                                ├── AI Fixer (LLM, ≤3 rounds)
                                └── Packager → Download
                              ↓
                         LLM Router (4 providers)
```

## New Models

Workspace, Conversation, Message, Artifact, Deployment

## New API

`/api/workspace/:id/*` — Files, Chat, Agent, SSE, Project detail
